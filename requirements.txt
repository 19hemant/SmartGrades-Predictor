# 🎓 SmartGrades Predictor  

SmartGrades Predictor is a **Machine Learning + Streamlit** application that predicts student marks based on the number of study hours.  
It uses a trained regression model to provide accurate grade forecasts in a simple and interactive interface.  

---

## 🚀 Features  
- Predicts student marks from study hours  
- User-friendly Streamlit interface  
- Motivational feedback based on predictions  
- Balloons 🎈 animation for excellent results  
- Real-time input validation (0–12 hours)  

---

## 🧑‍💻 How It Works  
1. Enter study hours (0–12)  
2. The trained ML model (`smp.pkl`) predicts marks out of 100  
3. App displays result with motivational message  

---

## 🛠️ Tech Stack  
- **Programming Language:** Python  
- **Frameworks & Libraries:** Streamlit, scikit-learn, Pandas, NumPy, Pickle  
- **Tools & Platforms:** Jupyter Notebook, VS Code, Git/GitHub
